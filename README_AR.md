# تطبيق المتجر (مبني من Figma)

تطبيق متجر مبني باستخدام Flutter وتم تنفيذ واجهاته ومخططاته اعتمادًا على تصميمات Figma الأصلية. يوفّر تطبيقًا أنيقًا ومتجاوبًا لعرض المنتجات، سلة التسوق، وحفظ المفضلات، وإدارة الطلبات.

## المميزات

- واجهة مستخدم نظيفة ومتجاوبة من تصميمات Figma
- عرض المنتجات، صفحة التفاصيل، إضافة إلى المفضلات والسلة
- تسجيل الدخول وإدارة الملف الشخصي
- استعراض المنتجات بحسب الفئات والبحث
- إشعارات وإدارة الطلبات

## مبني من

تم ترجمة التصميمات البصرية من Figma إلى عناصر واجهة في Flutter مع مراعاة المسافات والتوافق على شاشات مختلفة.

## لقطات الشاشة

توجد لقطات الشاشة والموارد في مجلد `screenshot/`. يمكن استبدالها أو إضافة صور للمعاينة.

## البدء

المتطلبات:

- Flutter SDK (القناة المستقرة)
- محرر مثل VS Code أو Android Studio

تشغيل المشروع:

1. استنساخ المستودع:

   git clone git@github.com:MohmadAliAlali/store_application_from_figma.git

2. الانتقال لمجلد المشروع:

   cd store_application_from_figma

3. تثبيت الحزم:

   flutter pub get

4. تشغيل التطبيق (مثال على محاكي أندرويد):

   flutter run

## بنية المشروع (ملخص)

- `lib/` — كود التطبيق (الشاشات، المتحكمات، النماذج، الأدوات المشتركة)
- `assets/`, `fonts/`, `icons/` — الموارد الثابتة
- مجلدات المنصات `android/`, `ios/`, `web/`، إلخ

المجلدات الرئيسية في `lib/`:

- `controller/` — منطق الحالة والتحكم
- `model/` — نماذج البيانات
- `view/` — الشاشات ومكوّنات الواجهة
- `core/` — الأنماط والمساعدة والملاحة المشتركة

## ملاحظات تطويرية

- التزم بأسلوب الكود الحالي عند إضافة أو تعديل المتحكمات والشاشات.
- راجع تصميم Figma عند تعديل الواجهات لضمان الاتساق البصري.
